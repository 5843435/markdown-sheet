\# 📘 プロダクト要件仕様書



\## \*\*Markdown Sheet — Excel-like Markdown Table Editor\*\*



---



\## 1. プロダクト概要



\*\*Markdown Sheet\*\* は、Markdown ファイル内の表（テーブル）を

→ \*\*Excel/スプレッドシートのようなインタラクティブ UI で編集できる Windows デスクトップアプリ\*\*



です。



既存の Markdown ビューア機能を拡張し、

Markdown のテーブルを

✔ その場で視覚編集

✔ 行・列操作

✔ コピー/貼り付け

✔ 改行・ドラッグ

✔ Git 互換のまま保存

できる体験を提供します。



目的：



\* Markdown + Git ワークフローを壊さない

\* 表の編集 UX を極限まで直観的にする

\* Excel のような感覚で Markdown を扱える



---



\## 2. 対象ユーザー



| カテゴリ           | ユーザー像                         |

| -------------- | ----------------------------- |

| エンジニア          | Markdown を要件定義／画面一覧／DB定義で使いたい |

| PM / PL        | 仕様テーブルを直接整理したい                |

| ドキュメント文化があるチーム | Git で表を管理したい                  |

| AI + ドキュメント活用者 | Markdown を解析対象にしたい            |



---



\## 3. 解決する課題



\### Problem



\* Markdown の表は\*\*テキストなので操作感が悪い\*\*

\* VS Code 以外のエディタで Markdown 表の編集がしにくい

\* Excel は Git で差分が扱いにくい

\* Markdown → Excel → また Markdown に戻す手間がかかる



\### Goal



Markdown 表を



> \*\*Excel のように直感的に操作でき、Git 管理もスムーズにできるツール\*\*



---



\## 4. 主な機能



---



\### 4-1. 表編集 UI



| 機能        | 説明                       |

| --------- | ------------------------ |

| 行／列の追加    | 行・列をドラッグ/コンテキストメニューで追加   |

| 行／列の削除    | 選択行/列を右クリックで削除           |

| セル編集      | Excel のようにセルをダブルクリックで編集  |

| Tab 移動    | Tab / Shift+Tab でセル移動    |

| 選択コピー     | 複数セルを選択してコピー             |

| 貼り付け      | Excel/Clipboard からペースト可能 |

| Undo/Redo | 操作の取り消し/やり直し             |

| 検索・置換     | 表内の検索/置換をシームレスに          |



---



\### 4-2. Markdown の入出力



| 読み込み | Markdown `.md` ファイル |

| 出力 | Markdown `.md` ファイル（Git と完全互換） |

| 保存 | 上書き保存／別名保存 |



---



\### 4-3. Markdown + ドキュメント構造



\* 見出しをまたいだ複数表の管理

\* 表のヘッダーも含め編集可能

\* 複数表の一括ナビゲーション



---



\### 4-4. Git との親和性



| 操作       | 対応                   |

| -------- | -------------------- |

| Git 差分   | Markdown のまま diff 表示 |

| Git コメント | ページ内コメント可能           |

| コミット     | 内蔵 Git 連携（拡張予定）      |



※ Git UIは初期 MVP では必須ではなく、後の拡張として優先



---



\## 5. UI/UX



\### 5-1. メイン画面



\* 左ペイン：ファイルツリー（プロジェクト）

\* 右ペイン：Markdown 表ビューア／エディタ

\* 上部：ツールバー（保存/Undo/Redo/追加/削除）

\* 編集中は \*\*Excel のようなカーソル移動とセルフォーカス\*\*



---



\### 5-2. 表操作



セルをダブルクリックで編集、

Tab で次セル移動、

右クリックでコンテキストメニュー：



```

右クリックメニュー：

・行追加（上／下）

・列追加（左／右）

・行削除

・列削除

・セル結合（オプション）

・クリップボード操作（コピー / カット / 貼り付け）

```



---



\## 6. データモデル



Markdown 表は下記のように解析/管理します：



\### Markdown パース構造



```

表 = {

&nbsp; headerCells: \[],

&nbsp; rowCells: \[

&nbsp;   \[],

&nbsp;   ...

&nbsp; ]

}

```



編集後は \*\*Markdown のパイプ区切りへ自動変換\*\*



---



\## 7. 拡張ポイント



\### 📌 Markdown + AI 連携



\* Copilot / Claude API 連携ボタン

\* 表の自動解析（列命名/ラベル付け）

\* 仕様書生成



例：



```

Markdown Sheet に読み込んだ 画面一覧 を

AI に解析させて API 仕様書を生成

```

\## 8. 非機能要件



| 要件    | 説明                           |

| ----- | ---------------------------- |

| OS    | Windows / macOS（初期は Windows） |

| 性能    | 大量行（10,000+）でも遅延なし           |

| データ保存 | UTF-8 Markdown               |

| 依存    | Electron / Tauri (軽量化)       |

| 開発    | TypeScript + React + WebView |

| 配布    | Windows EXE 版 + MSI          |
