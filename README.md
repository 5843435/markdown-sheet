# markdown-sheet

シンプルなMarkdownエディタ／Tauri + React プロジェクトです。

セットアップ（開発環境）:

```powershell
cd "C:\Users\akino\Desktop\claude\markdown-sheet"
npm install
npm run dev
```

デスクトップアプリのビルド（Tauri）:

```powershell
cd "C:\Users\akino\Desktop\claude\markdown-sheet"
npm run build
npx tauri build
```
